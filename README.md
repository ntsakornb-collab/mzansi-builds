# MzansiBuilds

A platform for developers to build in public, share projects, get help, and celebrate completions.

## Features

- User registration and login with password hashing
- Create projects with title, description, stage, and support needed
- Live feed showing all developer projects
- Comment on projects
- "Raise hand" button to request collaboration
- Update project stage (planning → in progress → completed)
- Celebration wall showing completed projects
- Green, white and black theme

## Tech Stack

- Python 3.10+
- Flask 3.0.0
- SQLite3
- HTML5 & CSS3
